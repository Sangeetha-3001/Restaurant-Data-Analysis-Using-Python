**Restaurant Data Analysis – Cognifyz Technologies Internship**

**Project Overview**
This project was completed as part of the Cognifyz Technologies Data Analysis Internship. The analysis was performed using Python to explore restaurant data, identify trends, and generate meaningful insights through data visualization and statistical analysis.

**Tools & Technologies**
Python
Pandas
NumPy
Matplotlib
Seaborn
Folium
Jupyter Notebook

**Level 1 Tasks**
**Task 1: Top Cuisines**
Identified the top three most common cuisines.
Calculated the percentage of restaurants serving each cuisine.
Visualized the results using a bar chart.
**Task 2: City Analysis**
Identified the city with the highest number of restaurants.
Calculated the average restaurant rating for each city.
Determined the city with the highest average rating.
**Task 3: Price Range Distribution**
Analyzed the distribution of restaurant price ranges.
Calculated the percentage of restaurants in each price category.
Created bar charts and histograms for visualization.
**Task 4: Online Delivery**
Calculated the percentage of restaurants offering online delivery.
Compared average ratings of restaurants with and without online delivery.
Visualized the comparison using charts.

**Level 2 Tasks**
**Task 1: Restaurant Ratings**
Analyzed the distribution of aggregate ratings.
Identified the most common rating range.
Calculated the average number of customer votes.
**Task 2: Cuisine Combination**
Identified the most common cuisine combinations.
Analyzed the average ratings of different cuisine combinations.
Visualized the results using bar charts.
**Task 3: Geographic Analysis**
Plotted restaurant locations using latitude and longitude.
Created an interactive map using Folium.
Identified geographical patterns in restaurant distribution.
**Task 4: Restaurant Chains**
Identified restaurant chains with multiple branches.
Analyzed their average ratings and customer votes.
Compared the popularity of the top restaurant chains.

**Project Outcomes**
Performed data cleaning and preprocessing.
Conducted exploratory data analysis (EDA).
Created meaningful visualizations.
Generated actionable insights from restaurant data.
Improved practical skills in Python and data analysis.
